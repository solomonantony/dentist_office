# Dentist Office Application
## Built by Solomon and Antony

This repository has the source code files of a web application for a trivial case. The application has a datatbase and a frontend.  The application is deliberately made to be simple to follow. Anyone starting to learn Python-SQLite-Streamlit for web development will find this useful. Use this [Google Slides](https://docs.google.com/presentation/d/183ZhubiTiTdcSrra0DZWQ8215bHAkWtHCSbQ9PjmsbY/edit?usp=sharing) for the notes.
